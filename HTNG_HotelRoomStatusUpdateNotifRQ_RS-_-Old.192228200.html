HTNG_HotelRoomStatusUpdateNotifRQ/RS

Message Description

Protel uses this message set to allow in room systems etc to receive and send various information to/from the PMS including updating of the guest room status. This message can be used to send Room status (cleaning) transfer via phone, PBX to PMS

  • PBX

  • HouseKeeping Apps

  • Mobile Devices

These tables show the attributes that protel uses. Please pay attention to attributes marked as available in SPE/MPE or protel AIR. Functionality enhancements are ongoing. So it is important to frequently check this site for updates. Unused elements/ attributes have been grayed out for clarity.
