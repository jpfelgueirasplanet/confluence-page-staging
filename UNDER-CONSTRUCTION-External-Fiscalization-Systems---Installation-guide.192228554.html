One way Installation

Screenshots

1

Fiscalisation needs to be activated and configured for the property (FiscalCodes, Invoice number ranges etc.)

To open the new protel Air system data, go to the Apps icon and click on System data.

Then, open the Financial Configuration tab in the left part of the screen, then on the Invoice tab where you can finally click the Fiscalization tab. The Fiscalization window opens.


Here you can manage the fiscal codes needed for fiscalization, and activate fiscalization in protel Air. A fiscal code is a unique code that is used to mark the fiscalized invoices. The fiscal code can be set up individually and according to the fiscal requirements of different countries (see below: Adding or editing fiscal codes).

2

Additional Config Settings

  • Configuration/currentProperty/propData/Fiscalization/Config/preSelectedFiscalizationMode: "BOTH"

    The following settings should be created automatically after the BOB Installation has been added as described above:

  • Configuration/currentProperty/propData/Interfaces/IO/Gateway/IO_InvoiceFiscalizeRQ/Disabled: false
  • Configuration/currentProperty/propData/Interfaces/IO/Gateway/IO_InvoiceFiscalizeRQ/TargetModule: "com.developer.module"
