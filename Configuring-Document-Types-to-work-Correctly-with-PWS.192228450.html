Configuring Document Types to work Correctly with PWS

  • PWS > 3.5.X
  • PWS < 3.5.X

PWS > 3.5.X

  1. Create the document types in system data in whichever language is required.


  2. From protel System Data>Reservation>OTA Mapping, select Document types from the dropdown box in the upper left corner.
  3. In the table on the left side, you will see the Document types already defined in your protel System Data. From the dropdown menu on the right-hand side, directly below OTA Preferences, select Document.
  4. To map the values from protel System Data with those taken from the OTA standard, please select your document type on the left side (1), select the corresponding OTA value on the right side (2), and click on Assign(3). In the table at the bottom of the dialog (4), you will now see the mapping completed.
  5. Proceed like this for all required document types.


PWS < 3.5.X

1. Create the document types in system data in whichever language required

2. Highlight the relevant Document type


3. Click on the 'World" Symbol under the budget button



4. Select English from the drop down list

5. Enter in the exact naming of the document type as defined in the OTA DocTypes Code list
