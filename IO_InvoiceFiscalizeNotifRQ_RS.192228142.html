1.0.

Authentication

Bearer Token

Message type

Asynchronous

Direction

Outbound

Message Description

Transmit invoice data & already assigned fiscal codes. Can be created for final invoices, information invoices and pro forma invoices depending on legal requirements and setup in pms.
