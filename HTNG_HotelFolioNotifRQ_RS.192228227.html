Message Description

protel I/O uses this request and response message pair to inform external system about folios details of a reservation. The HNTG_HotelFolioNotifRQ may be used to send a notifcication of one or more invoices for a reservation. This message may also be used if items on an invoice are modified or cancelled.

The Information can be used by the guest to review the charges incurred during their stay, or to get the information about the outstanding amount to be paid.
