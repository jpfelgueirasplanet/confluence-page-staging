HTNG_HotelStayUpdateNotifRQ/RS

Message Description

Protel use this message set to allow in room systems etc to receive various information from the PMS.
1- This message can be used to send Guest Stay data updates notifications from the PMS
2- Modification of DND status (button) PBX to PMS to various in-room systems that include but are not limited to:

  • PBX

  • Room environment controls

  • Mini-bar

  • In-room entertainment systems

Note!

These tables show the attributes that protel uses. Please pay attention to attributes marked as available in SPE/MPE and not available in protel AIR. Functionality enhancements in protel AIR are ongoing. So it is important to frequently check this site for updates.
