One way Installation


1

PWS Configuration

  • enable LiveMode and activate ToggleTrigger for required message types for 1-way connection

Message type

LiveMode

ToggleTrigger

OTA_HotelResNotifRQ

OTA_ProfileModifyRQ

2

Create a test reservation in the future (at least one year after the system date) & provide the reservation number and correlation ID to your integration partner

  • make a modification and finally a cancellation of your test reservation. Verify in Kibana that outgoing OTA_HotelResNotifRQ is answered by OTA_HotelResNotifRS and no error occurs. 
  • Modify the profile used in the test reservation and then verify in kibana that an outgoing OTA_ProfileModifyRQ is answered by a OTA_ProfileModifyRS and no error occurs

Please note that no bulk uploads are required for this integration partner and the integration is completed once they have verified the data is flowing
