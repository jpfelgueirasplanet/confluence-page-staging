pAir

OTA_HotelResNotifRQ

 

PMS send new, modified and cxld reservations (useful for walk-ins).

 

OTA_HotelAvailRQ

 

Obtaining a list of vacant and clean rooms that can be assigned to an arriving reservation. Only to find free rooms for check in.

 

OTA_ReadRQ/OTA_ResRetrieveRS (Reservation Lookup)

 

Requesting a reservation from the PMS using i.e. first name, last name, reservation ID.

 

IO_ResModifyNotifRQ

 

Assigning / Changing room numbers of reservations,
  • change attached GuestProfile
  • change invoice addresses
  • assign profiles
  • upselling 

 

HTNG_HotelCheckInNotifRQ

 

Check-in a specific reservation.

 

HTNG_HotelCheckOutNotifRQ

 

Check-out a specific reservation.

 

OTA_ProfileCreateRQ

 

Create new profiles  

 

OTA_ProfileModifyRQ

 

Sending updates to guest the profiles attached to a reservation, e.g. address modifications

 

IO_SystemStatusRQ

 

Requesting current PMS datein order to get the current arrival day.

 

IO_SystemDataRQ

 

Request the System Data from the PMS, which incl. the

Room and TAA details
