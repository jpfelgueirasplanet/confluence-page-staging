One way Installation


1

Create a test reservation with a test profile for the current business day and provide the reservation number to your integration partner

  • check in the reservation and look in Kibana that outgoing OTA_HotelResNotifRQ is answered by OTA_HotelResNotifRS and no error occurs
  • modify the check out date . Ask the integration partner if the change was reflected on their side as well
  • modify the test profile and add a Note that the test profile was changed. Ask the integration partner if the the change was reflected on their side as well
  • finally cancell your test reservation and inform the integration partner about this

Adding a note to the reservation

2

Create and start bulk uploads

  • for reservations (OTA_HotelResNotifRQ) for the next 1 year, starting with the installation date

In case you have issues with a stuck bulk upload, investigate why messages were not sent. As an option, delete the old job and create a new one.
