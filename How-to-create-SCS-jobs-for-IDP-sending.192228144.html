How to create SCS jobs for IDP sending

IDeaS is scheduling certain IDP times with the customers (in most of the cases). At these times the PWS needs to send additional statistics exports (delivered with OTA_HotelStatsNotifRQ).

The PWS needs to get triggered to start building and sending those messages. This triggering is done by the SCS (Scheduled Command Service).

That's why you need to create jobs in the SCS if IDeaS informs you about the IDP (Intra Day Processing) times. This article shows you how to do it step by step.

Creating SCS jobs


  1. Log in to: https://developer.protel.io

  2. Navigate to: Administration → Scheduled Command Service

  3. Choose Stage: PROD and select the property you would like to create the jobs for by name or HotelID (CloudID).


  4. Hit the 'Add Job' button

  5. Fill the required fields and click 'Save'
    (Timezone has been added to the screen - should always be the local timezone of the property the job is created for.)



    Value

    Comment

    MessageType

    OTA_HotelStatsNotifRQ


    Days

    • 45 Past (all PAST ReportCodes)
    • 365 Future (all CURRENT ReportCodes)

    This is the default - IDeaS would tell you if it needs to be different

    ReportCode

    • HTNG_PastDailyPropertyStatistics_v1
    • HTNG_PastDailyRoomTypeStatistics_v1
    • HTNG_PastDailySegmentRoomTypeStatistics_v1
    • HTNG_CurrentDailyPropertyStatistics_v1
    • HTNG_CurrentDailyRoomTypeStatistics_v1
    • HTNG_CurrentDailySegmentRoomTypeStatistics_v1

    You need to create one job per Report Code.

    Frequency

    Daily


    Start Date

    Choose the date at which the IDP export should start


    Start Time

    IDP Time scheduled by IDeaS


    Timezone

    Select the time zone where the property is located.

Deactivate SCS Jobs


You can also deactivate jobs and re-activate them again.

Active → switch is on (green) - to deactivate just click on the switch to turn them off



Re-activate → list the deactivated jobs and just click on the switch to turn them on again




Delete SCS Jobs


You can also delete obsolete jobs now - these could not be re-activated.
