Configuring Gender to work correctly with PWS

  • PWS Version > 3.6.X
  • PWS Version > 3.5.X
  • PWS Version < 3.5.X

PWS Version > 3.6.X

With a PWS Version > 3.6.X the Gender "unknown" can be configured as well. It is working in the same way as the "male" and "female"


PWS Version > 3.5.X

With a PWS Version > 3.5.x the Gender can be configured in the PWS directly.

Just open a IO Configuration Dialog, select the tab "Profiles" and configure the Gender.

Import! If no gender is until now configured in the System Data, the PWS will automatically create a new Gender for male and female.

PWS Version < 3.5.X

  1. Create Genders in System Data 

2. Highlight the first Gender

3. Click on the World Symbol 

4. Select English as the language and enter the required gender into the 'Text' field

5. Click 'New'


The PWS will look in the internat table for the required translation. This can be checked with this query

select * from internat where fieldx='restyp'
