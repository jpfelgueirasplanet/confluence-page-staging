Product Specification - Atomize

General information

Atomize is a Revenue Management System with a 2-way functionality for rate values.

Atomize I/O Integration is certified for both Protel On-Premise and Protel AIR.

Future Dates / Current Reports 

  • CurrentDailyPropertyStatistics - Hotel level (for the following 365 days in the future)
  • CurrentDailyRoomTypeStatistics - Room Type level
  • CurrentDailySegmentRoomTypeStatistics - Room Type / Market Code Combination


Historical Dates / Past Reports

  • PastDailyPropertyStatistics - Hotel level (for one day in the past)
  • PastDailyRoomTypeStatistics - Room Type level
  • PastDailySegmentRoomTypeStatistics - Room Type / Market Code Combination

FilterpAir: gov:sequences/protel/out/filters/DropPairMessageWhenIdsAndPairServiceInstalled.xml

  • This filter drops the pAir message in the ESB when the IDS is used. It only affects OTA_HotelAvailNotifRQ or OTA_HotelRatePlanNotifRQ:
  • The RQ message in pAir is generated but is dropped with log message "Message dropped due to applied restriction rules"
  • The message RQ from IDS is sent out to partner successfully

Filter onPrem: gov:sequences/protel/out/filters/AtomizeConcatUniqueIDRateElements.xml

  • changes multiple UniqueID Attribute values into a single one with a separator "@"
    This should be done for the @ID Element and @ID_Context Element

    Changes from
    <Rate>
    ...
    <UniqueID ID="ID:450" ID_Context="FixedPackage" Type="18"/>
    <UniqueID ID="ID:449" ID_Context="BookablePackage" Type="18"/>
    ...
    </Rate>
    To
    <Rate>
    ...
    <UniqueID ID="ID:450@ID:449" ID_Context="FixedPackage@BookablePackage" Type="18"/>
    ...
    </Rate>
