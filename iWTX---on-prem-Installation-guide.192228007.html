One way Installation


1

PWS Configuration

  • enable LiveMode and activate ToggleTrigger for required message types for 1-way connection

Message type

LiveMode

ToggleTrigger

OTA_HotelResNotifRQ

OTA_ProfileCreateRQ

IO_ResModifyRS

2

Configuring an IDS Instance for I/O: onPrem | How to configure a protel I/O IDS 

  • Add a new instance
  • Configure the Comserver URL
  • Add the URL for Export
  • Add the SQL database
  • Configure the Email Notification
  • Add the Token and Module Name in the Special Tab
  • Enable "fake" sync mode
  • Import
    • Import - Reservation - Common
      • Mode MUST be set to PULL and PUSH
    • Import Children
    • Notes - The integration partner requested to use Note 1 as default Note field. Also IDS memo notes are activated
  • Create DCS mapping / mpehotel/ Room Types /Rate Codes ( checked as fixed)/ Reservation Status/ Market/Source/Come/Hear Note: If automated matchings are used in the IDS configuration, then the IDS will create the matching for the respective category itself. But this will disable the option to do any adjustments on the auto-matched list. Thus, it is recommended to do the matchings manually

3

  • Deactivate in the IDS the Export data

  • Please activate in the IDS the "Children age ranges" (Config see sample screenshot)

  • Import reservation every 1 minute

4

Ask the integration partner to create a test reservation in the future (at least one year after the system date) & to provide you the reservation number. 

  • Ask the integration partner to make a modification to the reservation and to add a fix charge and a trace.
  • Ask the integration partner to cancel the reservation.


5

Update the protel SMP page with the information about the installed customer

Development SMP page / Development / Services / Prospect Hotels / Add
