IPA-732

ResModify | Inbound | Set Profile ID of arriving guest for anonymous slice

We will add a new use case to add additional arriving guests up to the guest count of the reservation .

IPA-717

Reservation | Inbound | Not possible to reinstate cancelled reservations

The cancelled reservation, now can be reinstated by sending an inbound reservation with following attributes:
OTA_HotelResNotifRQ>ResStatus="Modify"
and
OTA_HotelResNotifRQ / HotelReservations / HotelReservation >ResStatus="Reserved"

IPA-721

ResModify | Inbound | Create Routing Instruction by Template reference

We will add the use case to create a routing instruction using a predefined template. For this use case, the type needs to be "InvoiceRoutingTemplate" instead of "PaymentInstruction".

IPA-728

Reservation | Outbound | GuestCounts (on RoomRate level) empty for cancelled reservations

The GuestCounts element in RoomRate level is displaying for the cancelled reservations as well.
