Installation Guide - CKit

Step

Description

Image

1

 Download and copy file PWSToolSetup.xml to {ProtelPath}\System\Ckit\Xml


2

Go to SD -> Mics. -> Construction kit -> IO Support Tool –> 1. Initialize

 

3

In the opened Xsetup dialog, change the Hotel dropdownlist to ".Global", enter the full path and name of application exe and save the changes with "Set".


E.g. E:\Protels\protel31\PWS\protel.PWS.SupportSolution\protel.PWS.SupportSolution.exe

 

4

Go to SD -> Mics. -> Construction kit -> IO Support Tool -> 2. User Group access rights


5

Double click on one item to enable permission to a group of users or disable permission from the displayed window. Example, double click on “FO User” user group.

  • Note: Only users in an enabled permission group user can access function in Add-Ons

Installation Guide - protel I/O Support Tool

Step

Description

Image

1

 Download and copy file protel.PWS.SupportSolution.zip to {ProtelPath}\PWS

2

Call App via FO -> Add-Ons -> IO Support Tool -> Run Application:

 

3a.

When opening for the first time, the following information must be entered, to configure the Support Tool for the correct database 
- Server details
- SQL User
- SQL Password
- database
- PWS Path

3b.

As soon as 

- Server details
- SQL User
- SQL Password


are added, please check with the "TEST" Button if everything is correct, and working as expected

3c.

As soon as the SQL Connection is correct, you can select the database and add the PWS path

4

With "okay" the IO Support Tool opens

5

For each additional time, the login data is loaded from a config file. 

If you want to modify/adjust the config, the following config file needs to be deleted and you can start again with step 2.


E:\Protels\protel4\pws\protel.PWS.SupportSolution\SupportSolution.conf
