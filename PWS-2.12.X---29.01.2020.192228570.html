PWS-1026

IO

OTA_HotelResNotif outbound | 3rd Sharer only produces 1 reservation message rather than 3

If there are changes for a sharer reservation like attached a new sharer, cancel an existing sharer reservation, an OTA_HotelResNotifRQ for all attached sharer reservations is send out.

For reservation specific changes only the OTA_HotelResNotifRQ message is sent out for this specific reservation.

PWS-972

IO

OTA_HotelResNotif outbound | Exception if "no profile ID is existing"

If a profile for a reservation doesn't longer exist in the database "Kunden", we won't display the ResGuest Element anymore. We will send the message without ResGuest Element to the third-party system.
