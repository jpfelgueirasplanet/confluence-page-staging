One way Installation


1

Trigger exports

  • for availability (OTA_HotelInvCountNotifRQ)
    for the next 544 days
    Trigger the message using the API Trigger: OTA_HotelInvCountNotifRQ 

    Please use the API call for Room Type counts only.  No property counts are required. 


  • for restricitions (OTA_HotelAvailNotifRQ)
    for the next 544 days
    via IDS - trigger full export

  • for rates (OTA_HotelRatePlanNotifRQ)
    for the next 544 days
    via IDS - trigger full export

IMPORTANT

Please always cross check availability, rates and restrictions between PMS and Channel Manager with SmartHotel or the Hotel for a number of dates after completing this step to be sure that the systems are in Sync.

2

Data Validation Report - DQE

  • after checking if the bulk successfully finished, provide the integration partner the report
  • The Integration partner will process the DQE and send you a report afterwards
  • The report is to be found in the tab Reporting of the front office in pAir.
