IDS | Tax details for children

With IDS version → 524093 ← (TBA)  the children rate details can include tax details. Depending on the IDS configuration, the possibilities are:

  1. If IDS is configured to send only NET values, then

    • New attribute AdditionalGuestAmount/TaxInclusive="false" is shown to indicate that the value in AdditionalGuestAmount/@Amount is NET

      NET
      <AdditionalGuestAmounts>
      <AdditionalGuestAmount AgeQualifyingCode="7" Amount="22.00" CurrencyCode="EUR" MaxAge="3" MinAge="0" TaxInclusive="false"/>
      <AdditionalGuestAmount AgeQualifyingCode="8" Amount="44.00" CurrencyCode="EUR" MaxAge="7" MinAge="4" TaxInclusive="false"/>
      <AdditionalGuestAmount AgeQualifyingCode="9" Amount="66.00" CurrencyCode="EUR" MaxAge="12" MinAge="8" TaxInclusive="false"/>
      <AdditionalGuestAmount AgeQualifyingCode="19" Amount="88.00" CurrencyCode="EUR" MaxAge="17" MinAge="13" TaxInclusive="false"/>
      </AdditionalGuestAmounts>
  2. If IDS is configured to send only GROSS values

    • New attribute AdditionalGuestAmount/TaxInclusive="true" is shown to indicate that the value in AdditionalGuestAmount/@Amount is GROSS

    • New element Taxes with attributes Taxes/@Amount and Taxes/@CurrencyCode to indicate the actual value of the tax amount and resp currency per AdditionalGuestAmount 

      GROSS
      <AdditionalGuestAmounts>
      <AdditionalGuestAmount AgeQualifyingCode="7" Amount="20.00" CurrencyCode="EUR" MaxAge="3" MinAge="0" TaxInclusive="true">
      <Taxes Amount="1.82" CurrencyCode="EUR"/>
      </AdditionalGuestAmount>
      <AdditionalGuestAmount AgeQualifyingCode="8" Amount="40.00" CurrencyCode="EUR" MaxAge="7" MinAge="4" TaxInclusive="true">
      <Taxes Amount="3.64" CurrencyCode="EUR"/>
      </AdditionalGuestAmount>
      <AdditionalGuestAmount AgeQualifyingCode="9" Amount="50.00" CurrencyCode="EUR" MaxAge="12" MinAge="8" TaxInclusive="true">
      <Taxes Amount="5.45" CurrencyCode="EUR"/>
      </AdditionalGuestAmount>
      <AdditionalGuestAmount AgeQualifyingCode="19" Amount="80.00" CurrencyCode="EUR" MaxAge="17" MinAge="13" TaxInclusive="true">
      <Taxes Amount="7.27" CurrencyCode="EUR"/>
      </AdditionalGuestAmount>
      </AdditionalGuestAmounts>
  3. If IDS is configured to send both NET and GROSS values
    • New attribute AdditionalGuestAmount/TaxInclusive="false" is shown to indicate that the value in AdditionalGuestAmount/@Amount is NET

    • New element Taxes with attributes Taxes/@Amount and Taxes/@CurrencyCode to indicate the actual value of the tax amount and resp currency per AdditionalGuestAmount 

      NET and GROSS
      <AdditionalGuestAmounts>
      <AdditionalGuestAmount AgeQualifyingCode="7" Amount="18.18" CurrencyCode="EUR" MaxAge="3" MinAge="0" TaxInclusive="false">
      <Taxes Amount="1.82" CurrencyCode="EUR"/>
      </AdditionalGuestAmount>
      <AdditionalGuestAmount AgeQualifyingCode="8" Amount="36.36" CurrencyCode="EUR" MaxAge="7" MinAge="4" TaxInclusive="false">
      <Taxes Amount="3.64" CurrencyCode="EUR"/>
      </AdditionalGuestAmount>
      <AdditionalGuestAmount AgeQualifyingCode="9" Amount="54.55" CurrencyCode="EUR" MaxAge="12" MinAge="8" TaxInclusive="false">
      <Taxes Amount="5.45" CurrencyCode="EUR"/>
      </AdditionalGuestAmount>
      <AdditionalGuestAmount AgeQualifyingCode="19" Amount="72.73" CurrencyCode="EUR" MaxAge="17" MinAge="13" TaxInclusive="false">
      <Taxes Amount="7.27" CurrencyCode="EUR"/>
      </AdditionalGuestAmount>
      </AdditionalGuestAmounts>
