Message type

Message description 

The process being facilitated is that whenever a pre-defined, subscribed event takes place in the producing system (for example, whenever there is an update performed on the customer folio or after guest check out).


Note!

Each record in this table has been flagged to indicate whether an element or attribute is currently implemented in protel AIR or protel MPE/SPE (On-Prem). Functionality enhancements are ongoing, so it is important to check this documentation for regular updates.

Always the Reservation ID + Invoice ID needs to be stored by the external System. This ensures that existing Invoice IDs are not overwritten if they are not part of a message for a specific ReservationID.
