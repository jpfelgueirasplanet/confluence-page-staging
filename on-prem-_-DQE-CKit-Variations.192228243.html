DQE - Hotel Status Report

This revised version of the DQE-Hotel Status means that now only one CKit is required to obtain the different information, which previously required up to eight CKits to be deployed. Further enhancements now make it possible to view the reservations that make up the count and the ability to export the results into .xlsx, .txt or .CSV. 

Ckit


Installation Guide

Step

Description

Image

1

Download and copy the CKit file to {ProtelPath}\System\Ckit\Xml


2

Restart protel System Data and go to [protel SD > Misc. -> Construction Kit-> IO-DQE Reports].

3

Click on [Initialize] to initialize the CKit. This opens the X Setting dialog, where the extract information can be set. Change to Global settings

FileNameTemplate

text

The format of file name, currently we support:

$HotelShort$-> property short name

$hotel$ -> property name

$YFrom$ -> Year of from date

$MFrom$ -> Month of from date

$DFrom$ -> Day of from date

$YTo$ -> Year of to date

$MTo$ -> Month of to date

$DTo$ -> Day of to date

$YExp$ -> current year

$MExp$ -> current month

$DExp$ -> current day

$hourExp$ -> current hour

$minExp$  -> current minute

$secExp$ -> current second

User Guide

Step

Description

Image

1

Go to [protel FO -> Add-Ons ->IO DQE Reports -> IO | Hotel Status | DQE].

2

With the new version you can now select the following: 

  • Date range
  • Language
  • Day use included or not
  • Allotment blocks included or not
  • Revenue type

3

With the results on the screen you now have to options when right clicking on the data: 

  • Extract- Will extract the data to as per the XSettings set
  • Details - Will display the reservations for that day. 

4

Details for that day are displayed as follows 


5

To view the extracted file select IO |x Hotel Status | DQE Extract location

DQE - Daily Reservation Report

This revised version of the DQE-Daily Reservation means that now only one CKit is required to obtain the different information, which previously required up to six CKits to be deployed. Further enhancements now make it possible to select a data range for the results and the ability to export the results into .xlsx, .txt or .CSV.
