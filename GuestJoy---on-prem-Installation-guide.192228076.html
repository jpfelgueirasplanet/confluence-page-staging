OneClick- installation

For GuestJoy installations, please always use this way of I/O -  installation. 

Please note:
OneClick- installation is only replacing the installation in BOB. All necessary pre-setups are still requested as usual. 

After finishing the OneClick - Installation, please provide Integration Partner securely with token.


Please log in to the protel store: https://store.protel.io/#/ and search for GuestJoy in the search bar.

Make sure that you selected the hotel that is meant to get the product. Please compare the CloudID just to make sure. 


Click "Get".

Go through the wizard by clicking "next"

Accept the Terms of Service - click Next

Finish the installation:

IMPORTANT: please provide Installation Token securely to integration Partner e.G: via protel PrivateBin

4

In case of emergency:

Bob installation: one for on prem (if not existing) and one for the Integration  

  • provide the token corresponding to the integration partner on private Slack message or encrypted file by email and inform them by phone about the password.

Note: If you use the BoB installation, please always send an E-Mail to marketplace@protel.net in order to inform the team about the installation not being done via the protel store.

 

One way Installation


1

Create a test reservation in the future (at least one year after the system date) & provide the reservation number to your integration partner

  • make a modification and finally a cancellation of your test reservation. Verify in Kibana that outgoing OTA_HotelResNotifRQ is answered by OTA_HotelResNotifRS and no error occurs. 


2

Create and start bulk uploads

  • for reservations (OTA_HotelResNotifRQ)
    for all future reservations
  • any historic reservations only should be send in case the hotel requets it

BTS Jobs will be processed one by one as only one bulk per property is allowed at the same time. 

Bulks should be starting with Installation Date + 365 Days, except Hotel requests other date range.

In case you have issues with a stuck bulk upload, investigate why messages were not sent. As an option, delete the old job and create a new one. 


3

Data Validation Report - DQE

Detailed DQE is not required - confirm with partner the number of messages sent and received


4

Update the protel SMP page with the information about the installed customer

To be found here: 

Development SMP page / Development / Services / Prospect Hotels / Add
