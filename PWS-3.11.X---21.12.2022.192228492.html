PWS-2590

IO

Add disk space check

Added the possibility to enable a disk space check. If enabled (xsetup pws | io_CheckAvailableDiskSpace | 1 (global)) , PWS checks everey 4 hours if the available disk space is<10% of the overall disk capacity. If there is less capacity, an ifc error message is created, a log entry is created and the check interval is set to hourly. As soon as the capacity is back to >10%, the interval is set back to 4 hours.
