PWS-3382

IO

Availability | Inbound | to apply IsModify attribute to distinguish for availability outbound

Implemented a new attribute IsModify to the OTA_HotelAvailRQ message. If the inbound RQ is showing IsModify="true" the OTA_HotelAvailRS will not show the <Errors> element if only no available inventory is found, but return an empty result with the TPA_Extension <AvailabilityExtensions> filled. This will allow third party vendors to assign rooms of fully booked room types to existing reservations. If the new attribute is missing, blank or "false" the current logic will be used.
