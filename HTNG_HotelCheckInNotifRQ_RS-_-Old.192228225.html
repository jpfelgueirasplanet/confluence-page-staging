HTNG_HotelCheckInNotifRQ/RS

Message Description

Protel uses this message set to receive check in notifications from Kiosk systems.


Note!

These tables show the attributes that protel uses. Please pay attention to attributes marked as available in on prem and not available in protel AIR. Functionality enhancements are ongoing. So it is important to frequently check this site for updates.

OnPrem:

With OnPremise the Hotel can configured with Rooms Status are allowed to be checked-in and which not. The Hotel can decide between the following Room Status.

  • Clean + Checked
  • Only Clean
  • Only Checked
