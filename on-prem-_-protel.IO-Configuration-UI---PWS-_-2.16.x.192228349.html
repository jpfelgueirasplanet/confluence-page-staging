With PWS version 2.16.x we have introduced an extended protel.IO configuration via PWS UI. 

You are now able to configure most of the used XSetUps via the PWS dialog instead of entering them in XSetUp screen in protel on prem manually (copy & paste).

The configuration is available with a double-click on protel.WebServices.exe and choosing Settings/ProtelIO configuration.


Please see below a description of every single tab in this dialogue.
All XSetUps marked as 'local' can be saved per property (relevant mpe property needs to be selected via drop down on top of each of the tabs.

If you hover the mouse over the label of a setting, you'll get a tooltip which shows the affected XSetUp.



Installations

There are no changes on this tab. 


General settings

We gathered all settings from the general preparation on this extended tab.
Starting with PWS version 2.16.x it is therefore not necessary anymore to activate these settings - there is a default value hard coded.

We also implemented a new feature to reset all settings of this tab to default settings (see the button next to 'OK') - "Reset to default settings".
This will give you the "standard" configuration including the standard URLs (PROD environment!)!
